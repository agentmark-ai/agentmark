# Mastra Adapter

> [!WARNING]
> **DEPRECATED.** `@agentmark-ai/mastra-v0-adapter` is deprecated: it receives no new features, and removal from this repository is planned. Use [`@agentmark-ai/fallback-adapter`](../fallback-adapter) — the default adapter path — instead.

This is an adapter for Mastra. It allows you to use AgentMark prompts with Mastra agents.
