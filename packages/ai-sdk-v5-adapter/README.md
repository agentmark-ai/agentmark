# AI V5 Adapter

> [!WARNING]
> **DEPRECATED.** `@agentmark-ai/ai-sdk-v5-adapter` is deprecated: it receives no new features, and removal from this repository is planned. Use [`@agentmark-ai/fallback-adapter`](../fallback-adapter) — the default adapter path — instead.

This is an adapter for the AI SDK v5. It allows you to use AgentMark prompts with the Vercel AI SDK v5.
