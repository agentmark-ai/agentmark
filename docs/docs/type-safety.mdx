# Type Safety

**WARNING!** Type safety is still actively being worked on. It is not feature complete yet.

AgentMark supports type-safety with just a few short steps.

![Type Safety](/img/type-safety.gif)

## Ading Type Safety

### Adjust your tsconfig

**Note**: It is recommended to use a separate `tsconfig.json` file for your AgentMark files.

```json
{
  "compilerOptions": {
    ...,
    "allowJs": true
  },
  "mdx": {
    "checkMdx": true
  }
}
```

### Add JSDOC types to define props for your agentmark files
```jsx
{/**
  * @typedef Props
  * @property {string} name - Who to greet.
  */
}

# Hello {props.name}
```

### Add types for your filters/tags

Create a `types/global.d.ts` file and define your own custom filters/tags.

```tsx
import type { BaseMDXProvidedComponents, FilterFunction } from '@puzzlet/agentmark';

declare global {
  const myCustomFilter: FilterFunction<string, string>;

  interface MDXProvidedComponents extends BaseMDXProvidedComponents {
    MyCustomTag: React.FC<MyCustomTagProps>;
  }
}
export {};
```
